# AMR 音频文件播放 DEMO

首先请全局安装一个静态资源服务器，如 [http-server](https://www.npmjs.com/package/http-server) 或者 [vite](https://vitejs.dev/guide/)（此处推荐 `vite`，它支持热更新，本地修改的内容会实时推送到浏览器）。

- http-server:
  ```bash
  sudo yarn add http-server -g

  http-server
  ```

- vite:
  ```bash
  sudo yarn add vite -g

  vite
  ```
